# Contributing to LifeZen Support

Thank you for helping improve LifeZen.

This repository is focused on product feedback, support, waste-sorting examples, and public improvement ideas.

Useful contributions:

- report wrong or unclear sorting recommendations;
- add municipality-specific corrections;
- suggest clearer wording in German or English;
- report accessibility issues;
- share confusing edge cases with photos or descriptions;
- suggest better privacy-preserving flows.

Before opening an issue, please remove private information from screenshots and descriptions.

LifeZen should be practical, calm, privacy-first, and honest about uncertainty when local disposal rules differ.
