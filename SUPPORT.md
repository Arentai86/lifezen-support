# Support

Use GitHub Issues for LifeZen support and feedback:

https://github.com/Arentai86/lifezen-support/issues

Please include:

- what item or material you scanned;
- what answer you expected;
- what LifeZen answered instead;
- city/municipality if local rules matter;
- screenshots if useful.

Do not post private data, exact home addresses, personal IDs, payment information, private documents, or API keys in public issues.

For general OpenZen information:

https://openzen.info
